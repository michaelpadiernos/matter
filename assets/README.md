# Assets

This directory contains all browser-ready files served to the front end by the Matter theme — compiled CSS, bundled JavaScript, fonts, and images. Everything here is build output; do not edit these files directly. Make changes in `source/` and let the Gulp pipeline regenerate them.

## Structure

| Directory  | Contents                                                                 |
| ---------- | ------------------------------------------------------------------------ |
| `css/`     | Compiled CSS output from the PostCSS pipeline                            |
| `js/`      | Bundled JavaScript output from the Gulp pipeline                         |
| `fonts/`   | Web font files (Roboto Regular, Mono, and Slab) served directly          |
| `images/`  | Static image assets, including the Font Awesome icon library             |
